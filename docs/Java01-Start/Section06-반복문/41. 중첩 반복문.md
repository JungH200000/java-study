# 중첩 반복문

**정의**: **중첩 반복문**은 반복문 안에 **다른 반복문을 포함**하는 형태다.
**요약**: 바깥 반복이 한 번 돌 때마다 **안쪽 반복이 전부 실행**된다. (`for`, `while` 모두 가능)

---

## 예제: `Nested1`

```java
package loop;

public class Nested1 {
    public static void main(String[] args) {
        for (int i = 0; i < 2; i++) { // 외부 for: i = 0, 1 (총 2회)
            System.out.println("외부 for 시작 i:" + i);

            for (int j = 0; j < 3; j++) { // 내부 for: j = 0, 1, 2 (각 i마다 3회)
                System.out.println("-> 내부 for " + i + "-" + j);
            }

            System.out.println("외부 for 종료 i:" + i);
            System.out.println(); // 라인 구분을 위해 실행
        }
    }
}
```

### 실행 결과

```
외부 for 시작 i:0
-> 내부 for 0-0
-> 내부 for 0-1
-> 내부 for 0-2
외부 for 종료 i:0
외부 for 시작 i:1
-> 내부 for 1-0
-> 내부 for 1-1
-> 내부 for 1-2
외부 for 종료 i:1
```

### 동작 설명

* **외부 for**는 `i = 0, 1`로 **2번** 반복된다.
* **내부 for**는 `j = 0, 1, 2`로 **3번** 반복된다.
* 외부 1회당 내부가 3회 실행되므로, 내부 출력은 **2 × 3 = 6회** 수행된다.
