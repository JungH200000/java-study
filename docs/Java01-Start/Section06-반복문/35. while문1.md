# while문 1

**정의**: `while`문은 **조건식이 참(`true`)인 동안 코드 블록을 반복 실행**한다.
**요약**: “조건 → 실행 → 재검사” 순환을 통해 같은 작업을 여러 번 수행한다.

---

## 기본 형태

```java
while (조건식) {
    // 반복해서 실행할 코드
}
```

* **조건식 검사**: 참이면 블록 실행, 거짓이면 `while`문 종료.
* **반복 흐름**: 블록이 끝나면 **다시 조건식으로 돌아가** 참/거짓을 재검사한다. (조건이 계속 참이면 반복 지속)

---

## 예시: 1을 한 번씩 더해 총 3번 더하기 (`While1_2`)

```java
package loop;

public class While1_2 {
    public static void main(String[] args) {
        int count = 0;                      // 초기값: 0

        while (count < 3) {                 // 조건식: count가 3보다 작은 동안 반복
            count++;                        // 핵심 작업: count를 1 증가
            System.out.println("현재 숫자는:" + count); // 현재 값 출력
        }
    }
}
```

### 출력

```
현재 숫자는:1
현재 숫자는:2
현재 숫자는:3
```

### 실행 흐름

* while문 실행1: `count = 0` → `count < 3` 참 → 블록 실행 → `count`가 `1`이 되고 출력
* while문 실행2: `count = 1` → `count < 3` 참 → 블록 실행 → `count`가 `2`가 되고 출력
* while문 실행3: `count = 2` → `count < 3` 참 → 블록 실행 → `count`가 `3`이 되고 출력
* while문 실행4: `count = 3` → `count < 3` 거짓 → **반복 종료**

> 참고: 위 예제에서 `while (count < 100)`으로 바꾸면 **코드 블록이 100번 반복**된다.
