# while문 2

**정의**: `while`문은 **조건이 참인 동안 동일한 코드 블록을 반복 실행**한다.
**요약**: “조건 검사 → 블록 실행 → 조건 재검사” 흐름으로 반복 횟수를 유연하게 조절한다.

---

## 1) 문제: 1부터 하나씩 증가하는 수를 3번 더하기 (1 ~ 3)

먼저 `while` 없이 단순하게 풀어본다.

### 예제: `While2_1`

```java
package loop;

public class While2_1 {
    public static void main(String[] args) {
        int sum = 0;

        sum = sum + 1; // sum(0) + 1 -> sum(1)
        System.out.println("i=" + 1 + " sum=" + sum);

        sum = sum + 2; // sum(1) + 2 -> sum(3)
        System.out.println("i=" + 2 + " sum=" + sum);

        sum = sum + 3; // sum(3) + 3 -> sum(6)
        System.out.println("i=" + 3 + " sum=" + sum);
    }
}
```

#### 출력 결과

```
i=1 sum=1
i=2 sum=3
i=3 sum=6
```

* 정답은 맞지만, **변경에 취약**하다(예: 시작값이나 반복 횟수가 바뀌면 코드를 많이 고쳐야 함).

---

## 2) 변수로 변경에 유연하게 만들기

요구사항 변경: **10부터 하나씩 증가하는 수를 3번 더하기 (10 ~ 12)**
→ 매번 상수를 바꾸는 대신, **변수 `i`**로 변경 구간의 시작값을 관리한다.

### 예제: `While2_2`

```java
package loop;

public class While2_2 {
    public static void main(String[] args) {
        int sum = 0;
        int i = 1; // 시작값

        sum = sum + i; // sum(0) + i(1) -> sum(1)
        System.out.println("i=" + i + " sum=" + sum);

        i++; // i = 2
        sum = sum + i; // sum(1) + i(2) -> sum(3)
        System.out.println("i=" + i + " sum=" + sum);

        i++; // i = 3
        sum = sum + i; // sum(3) + i(3) -> sum(6)
        System.out.println("i=" + i + " sum=" + sum);
    }
}
```

#### 출력 결과(`i=1`)

```
// i=1
i=1 sum=1
i=2 sum=3
i=3 sum=6
```

* 이제 **시작값 `i`만 바꾸면** 같은 로직으로 다른 구간도 계산 가능.

#### 출력 결과(`i=10`로 변경)

```
// i=10
i=10 sum=10
i=11 sum=21
i=12 sum=33
```

> **좋은 코드의 기준**: 변경이 생겼을 때 **수정 범위가 작을수록** 좋다.

---

## 3) 새로운 요구사항: `i`부터 `endNum`까지 더하기 (i ~ endNum)

* 예) `i=1, endNum=3` → 총 **3번** 더하기
* 예) `i=1, endNum=10` → 총 **10번** 더하기
* 예) `i=10, endNum=12` → 총 **3번** 더하기

### (단순 풀이) 반복 코드를 직접 작성 — `While2_3`

```java
package loop;

public class While2_3 {
    public static void main(String[] args) {
        int sum = 0;
        int i = 1;
        int endNum = 3;

        sum = sum + i;
        System.out.println("i=" + i + " sum=" + sum);
        i++;

        sum = sum + i;
        System.out.println("i=" + i + " sum=" + sum);
        i++;

        sum = sum + i;
        System.out.println("i=" + i + " sum=" + sum);
        i++;
    }
}
```

#### 실행 결과

```
i=1 sum=1
i=2 sum=3
i=3 sum=6
```

여기서 실제로 **반복되는 코드 조각**은 아래 3줄이다. `endNum`이 커질수록 이 3줄을 **많이 복붙**해야 하므로 비효율적이다.

```java
sum = sum + i;
System.out.println("i=" + i + " sum=" + sum);
i++;
```

## 4) `while`로 반복 횟수를 유연하게 제어 — `While2_3` (코드 변경)

```java
package loop;

public class While2_3 {
    public static void main(String[] args) {
        int sum = 0;
        int i = 1;
        int endNum = 3;

        while (i <= endNum) {                 // i가 endNum이 될 때까지 반복
            sum = sum + i;                    // 누적 합산
            System.out.println("i=" + i + " sum=" + sum); // 진행 상황 출력
            i++;                              // 다음 수로 이동
        }
    }
}
```

### 반복 횟수 정하기

* `i=1, endNum=3`이면 **3번** 반복: `1 → 2 → 3`
* `i=3, endNum=4`이면 **2번** 반복: `3 → 4`

### while문 작성 요령

* `while (i <= endNum)` 조건으로 **반복 구간의 끝값**을 제어한다.
* `i`가 `endNum`보다 **크면 종료**된다.

### 실행 결과

```
// i=1, endNum=3
i=1 sum=1
i=2 sum=3
i=3 sum=6

// i=1, endNum=10
i=1 sum=1
i=2 sum=3
i=3 sum=6
i=4 sum=10
i=5 sum=15
i=6 sum=21
i=7 sum=28
i=8 sum=36
i=9 sum=45
i=10 sum=55

// i=10, endNum=12
i=10 sum=10
i=11 sum=21
i=12 sum=33
```

## 5) 흐름 이해 (i=1, endNum=3)

* `i=1, endNum=3` → **조건 만족**, `i=1, sum=1` 출력
* `i=2, endNum=3` → **조건 만족**, `i=2, sum=3` 출력
* `i=3, endNum=3` → **조건 만족**, `i=3, sum=6` 출력
* `i=4, endNum=3` → **조건 불만족**, **while 종료**

### 실행 코드 분석

```java
sum(0), i(1), endNum(3)

// 루프 1
while (i(1) <= endNum(3)) -> true
sum(0) + i(1) -> sum(1)
i(1)++ -> i(2)

// 루프 2
while (i(2) <= endNum(3)) -> true
sum(1) + i(2) -> sum(3)
i(2)++ -> i(3)

// 루프 3
while (i(3) <= endNum(3)) -> true
sum(3) + i(3) -> sum(6)
i(3)++ -> i(4)

// 루프 4
while (i(4) <= endNum(3)) -> false
```
