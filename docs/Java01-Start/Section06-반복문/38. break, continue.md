# break, continue

**정의**: `break`는 **반복문을 즉시 종료**하고, `continue`는 **현재 반복의 남은 코드를 건너뛰고 다음 반복**으로 진행한다.
`while`, `do-while`, `for` **모든 반복문에서 사용 가능**하다.

---

## 1) `break`

### 기본 형태

```java
while (조건식) {
    코드1;
    break; // 즉시 while문 종료로 이동한다.
    코드2; // 실행되지 않는다.
}
// while문 종료
```

* `break`를 만나면 **바로 반복문을 빠져나간다**. 따라서 `코드2`는 실행되지 않는다.

### 예제: 누적 합이 10보다 처음으로 큰 시점 찾기 — `Break1`

문제: **1부터 시작해 숫자를 계속 누적해 더하다가** 합계가 **10보다 처음으로 큰 값**은 얼마인가?

```java
package loop;

public class Break1 {
    public static void main(String[] args) {
        int sum = 0;
        int i = 1;

        while (true) {                   // 조건이 항상 참 → 무한 반복 (중간에 break로 탈출)
            sum += i;
            if (sum > 10) {              // 합이 10보다 크면
                System.out.println("합이 10보다 크면 종료: i=" + i + " sum=" + sum);
                break;                   // 즉시 while문 종료
            }
            i++;
        }
    }
}
```

#### 실행 결과

```
합이 10보다 크면 종료: i=5 sum=15
```

* `while (true)`는 **항상 참**이라 무한 반복이지만, **조건을 만족하면 `break`로 탈출**한다.

---

## 2) `continue`

### 기본 형태

```java
while (조건식) {
    코드1;
    continue; // 즉시 조건식으로 이동한다.
    코드2;    // 실행되지 않는다.
}
```

* `continue`를 만나면 **현재 반복의 나머지 부분을 건너뛰고** 곧바로 **조건식 검사**로 이동한다. 조건이 참이면 다음 반복을 실행한다.

### 예제: 1부터 5까지 출력하되 3은 건너뛰기 — `Continue1`

문제: 1부터 5까지 숫자를 출력하는데, 숫자가 3일 때는 출력을 건너뛰어야 한다.

```java
package loop;

public class Continue1 {
    public static void main(String[] args) {
        int i = 1;

        while (i <= 5) {
            if (i == 3) {   // 3일 때는 출력 건너뛰기
                i++;        // 다음 값으로 이동
                continue;   // 즉시 조건식으로 이동 (아래 출력문 건너뜀)
            }
            System.out.println(i);
            i++;
        }
    }
}
```

#### 실행 결과

```
1
2
4
5
```

* `i == 3`일 때는 `i`를 증가시키고 `continue`로 **출력을 건너뛰어** 바로 조건식으로 이동한다.
