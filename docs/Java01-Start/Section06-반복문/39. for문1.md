# for문 1

**정의**: `for`문은 **반복 횟수가 정해져 있을 때** 코드를 반복 실행하는 구문이다.
반복에 필요한 **초기화·조건 검사·증가(감소)** 를 한 줄에 모아 읽기 쉽다.

---

## 기본 구조와 실행 순서

```java
for (/* 1.초기식 */ ; /* 2.조건식 */ ; /* 4.증감식 */) {
    // 3.코드
}
```

**실행 순서**

1. **초기식** 실행(딱 한 번): 보통 반복에 사용할 변수를 선언·초기화한다.
2. **조건식** 검사: 참이면 블록(코드)을 실행, 거짓이면 `for`문 종료.
3. **코드** 실행: 반복 본문 수행.
4. **증감식** 실행: 보통 카운터 변수를 증가/감소.
5. 다시 **2. 조건식**으로 돌아가 위 과정을 반복한다. (조건이 거짓이 될 때까지)

---

## 예: 1부터 10까지 출력

반복 횟수가 명확할 때 `for`문을 쓰면 코드가 간결해진다.

```java
for (int i = 1; i <= 10; i++) {
    System.out.println(i); // i를 1부터 10까지 출력
}
```

* `int i = 1`(초기식) → `i <= 10`(조건식) → `System.out.println(i)`(코드) → `i++`(증감식) → 조건 재검사
* 조건이 거짓(`i <= 10`이 거짓)이 되는 순간 반복을 종료한다.

### 전체 클래스: `For1`

```java
package loop;

public class For1 {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {   // 1~10까지 순서대로 출력
            System.out.println(i);
        }
    }
}
```

## 문제: i부터 endNum(마지막 수)까지 누적 합 (i ~ endNum 더하기)

`for`문을 사용해서 합계를 구한다.

### 예: `For2`

```java
package loop;

public class For2 {
    public static void main(String[] args) {
        int sum = 0;
        int endNum = 3;

        for (int i = 1; i <= endNum; i++) {        // i=1부터 endNum까지 반복
            sum = sum + i;                          // 누적 합
            System.out.println("i=" + i + " sum=" + sum); // 진행 상황 출력
        }
    }
}
```

#### 출력 결과

```
i=1 sum=1
i=2 sum=3
i=3 sum=6
```

## for vs while (동일 문제 비교)

같은 문제를 `while`로 풀었던 코드와 비교해 보자.

### 참조: `While2_3`

```java
package loop;

public class While2_3 {
    public static void main(String[] args) {
        int sum = 0;
        int i = 1;
        int endNum = 3;

        while (i <= endNum) {                         // 조건이 참인 동안 반복
            sum = sum + i;                            // 누적 합
            System.out.println("i=" + i + " sum=" + sum);
            i++;                                      // 카운터 증가(반복 제어)
        }
    }
}
```

**비교 포인트**

* `for (int i = 1; i <= endNum; i++)` 처럼 **초기화·조건·증감을 한 줄에 모아** 카운터 변수를 명확히 드러낸다.
* 덕분에 **루프 제어 코드**와 **업무 로직**을 쉽게 구분할 수 있다.
* 반면 `while`은 `i` 선언과 `i++` 증가가 **본문 곳곳에 분산**될 수 있어 흐름 파악이 비교적 어렵다.
