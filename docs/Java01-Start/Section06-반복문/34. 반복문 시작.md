# 반복문 시작 (Java)

반복문은 **특정 코드를 여러 번 반복 실행**할 때 사용한다.

자바는 세 가지 반복문을 제공한다: **`while`**, **`do-while`**, **`for`**. 같은 동작을 여러 번 작성하지 않고 **중복을 줄여 코드 가독성과 유지보수성**을 높인다.

---

## 왜 반복문이 필요한가

아래 예제는 `count` 값을 **1씩 세 번 증가**시키며, 증가할 때마다 현재 값을 출력한다. 같은 코드가 반복되므로, 이런 상황에서 반복문이 필요해진다.

### 예시: `While1_1`

```java
package loop;

public class While1_1 {
    public static void main(String[] args) {
        int count = 0;                     // 1) 초기값 설정

        count = count + 1;                 // 2) +1
        System.out.println("현재 숫자는:" + count); // 출력: 1

        count = count + 1;                 // 3) +1
        System.out.println("현재 숫자는:" + count); // 출력: 2

        count = count + 1;                 // 4) +1
        System.out.println("현재 숫자는:" + count); // 출력: 3
    }
}
```

#### 출력

```
현재 숫자는:1
현재 숫자는:2
현재 숫자는:3
```

* 위 코드는 **동일한 패턴(증가 → 출력)**이 **3번 반복**된다.
* 최종적으로 `count` 값은 **3**이 된다.

---

## 개선: 증감 연산자 `++`

`count = count + 1`은 **증감 연산자 `++`**로 더 간단하게 표현할 수 있다. 아래는 같은 동작을 **간결하게** 바꾼 코드 조각이다.

```java
// 개선
count++;
System.out.println("현재 숫자는:" + count);

count++;
System.out.println("현재 숫자는:" + count);

count++;
System.out.println("현재 숫자는:" + count);
```

* `count++`는 `count = count + 1`과 **동일한 효과**를 낸다.
* 여전히 **같은 형태의 코드가 반복**되고 있어, 반복문으로 바꾸면 더 깔끔해진다.
