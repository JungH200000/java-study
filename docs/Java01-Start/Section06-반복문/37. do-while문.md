# do-while문

**정의**: `do-while`문은 **조건과 상관없이 코드 블록을 최소 1회 실행**한 뒤, **조건식을 검사**하여 반복을 이어가는 구문이다.
**요약**: “실행 → 조건 검사” 순서이기 때문에 **최초 한 번은 반드시 실행**된다.

---

## 기본 구조

```java
do {
    // 코드
} while (조건식);
```

* 블록을 **먼저 실행**하고, 그 다음 **조건식**을 검사한다.
* 조건이 **참**이면 다시 블록으로 돌아가고, **거짓**이면 반복을 종료한다.

---

## while문과의 대비 예제

먼저 `while`을 사용했을 때를 보자. 초기값이 조건을 **만족하지 않으면 아무 것도 실행되지 않음**을 확인한다.

### 예제: `DoWhile1` (while 사용)

```java
package loop;

public class DoWhile1 {
    public static void main(String[] args) {
        int i = 10;

        while (i < 3) {                         // 조건이 거짓(false) → 블록 미실행
            System.out.println("현재 숫자는:" + i);
            i++;
        }
    }
}
```

#### 출력 결과

```
// 없음
```

* `i = 10`이므로 `i < 3`은 **거짓** → **한 번도 실행되지 않음**.

---

## do-while로 최소 1회 실행 보장

같은 상황에서 `do-while`을 사용하면, **먼저 한 번 실행**된 뒤 조건을 검사한다.

### 예제: `DoWhile2` (do-while 사용)

```java
package loop;

public class DoWhile2 {
    public static void main(String[] args) {
        int i = 10;

        do {
            System.out.println("현재 숫자는:" + i); // 최소 1회 출력
            i++;
        } while (i < 3);                          // i가 11이 되어 거짓 → 종료
    }
}
```

#### 출력 결과

```
현재 숫자는:10
```

* 블록 실행 후 `i = 11`, `i < 3`은 **거짓** → **반복 종료**.
* “한 번은 꼭 실행해야 하는 경우”에 `do-while`을 사용한다.
