# 2차원 배열 — 시작

## 개념 정리

* **정의:** 2차원 배열은 **행(row)** 과 **열(column)** 로 구성된 배열입니다.
* **한 문장 요약:** `arr[행][열]` 형식으로 **두 개의 인덱스**로 접근합니다.
* 사용법은 **대괄호 `[]`가 하나 더 붙는 것**을 제외하면 1차원 배열과 같습니다.

## 선언·생성과 인덱싱

* 선언·생성: `int[][] arr = new int[2][3];` → **2행 3열**
* 접근: `arr[1][2]` → **1행**에서 **2열** 요소

|     | [0] | [1] | [2] |
|-----|-----|-----|-----|
| [0] | 1   | 2   | 3   |
| [1] | 4   | 5   | 6   |

표의 배열에 담긴 데이터를 텍스트로 정리하면 다음과 같습니다.

* `arr[0][0] = 1`, `arr[0][1] = 2`, `arr[0][2] = 3`
* `arr[1][0] = 4`, `arr[1][1] = 5`, `arr[1][2] = 6`

---

## ArrayDi0

```java
package array;

public class ArrayDi0 {
    public static void main(String[] args) {
        // 2x3 2차원 배열을 만든다.
        int[][] arr = new int[2][3]; // 행(row), 열(column)

        arr[0][0] = 1; // 0행, 0열
        arr[0][1] = 2; // 0행, 1열
        arr[0][2] = 3; // 0행, 2열
        arr[1][0] = 4; // 1행, 0열
        arr[1][1] = 5; // 1행, 1열
        arr[1][2] = 6; // 1행, 2열

        // 0행 출력
        System.out.print(arr[0][0] + " "); // 0열 출력
        System.out.print(arr[0][1] + " "); // 1열 출력
        System.out.print(arr[0][2] + " "); // 2열 출력
        System.out.println();              // 한 행이 끝나면 줄바꿈

        // 1행 출력
        System.out.print(arr[1][0] + " "); // 0열 출력
        System.out.print(arr[1][1] + " "); // 1열 출력
        System.out.print(arr[1][2] + " "); // 2열 출력
        System.out.println();              // 한 행이 끝나면 줄바꿈
    }
}
```

> 설명
>
> * **값 대입:** 행→열 순서로 `arr[행][열]`에 값을 넣습니다.
> * **값 출력:** 0행의 0·1·2열을 차례대로 출력한 뒤 줄바꿈, 다음에 1행의 0·1·2열을 출력합니다.

### 실행 결과

```
1 2 3 //[0][0], [0][1], [0][2]
4 5 6 //[1][0], [1][1], [1][2]
```
