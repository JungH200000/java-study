# 2차원 배열 - 리펙토링1: 구조 개선(행 출력 반복)

## 구조 변경 아이디어

코드 구조를 보면 **행마다 같은 패턴의 출력**이 반복됩니다.

```java
// 0행 출력
System.out.print(arr[0][0] + " "); // 0열 출력
System.out.print(arr[0][1] + " "); // 1열 출력
System.out.print(arr[0][2] + " "); // 2열 출력
System.out.println();              // 한 행이 끝나면 라인 변경

// 1행 출력
System.out.print(arr[1][0] + " "); // 0열 출력
System.out.print(arr[1][1] + " "); // 1열 출력
System.out.print(arr[1][2] + " "); // 2열 출력
System.out.println();              // 한 행이 끝나면 라인 변경
```

차이는 **행 인덱스(0 ↔ 1)** 뿐이므로, 다음처럼 `row`만 바꿔가며 **반복**하면 됩니다.

```java
// row를 0, 1로 변경하면서 다음 코드를 반복
System.out.print(arr[row][0] + " "); // 0열 출력
System.out.print(arr[row][1] + " "); // 1열 출력
System.out.print(arr[row][2] + " "); // 2열 출력
System.out.println();                // 한 행이 끝나면 라인 변경
```

### ArrayDi1

```java
package array;

public class ArrayDi1 {
    public static void main(String[] args) {
        // 2x3 2차원 배열을 만듭니다.
        int[][] arr = new int[2][3]; // 행(row), 열(column)

        arr[0][0] = 1; // 0행, 0열
        arr[0][1] = 2; // 0행, 1열
        arr[0][2] = 3; // 0행, 2열
        arr[1][0] = 4; // 1행, 0열
        arr[1][1] = 5; // 1행, 1열
        arr[1][2] = 6; // 1행, 2열

        for (int row = 0; row < 2; row++) {   // 행을 0 → 1 순서로 반복
            System.out.print(arr[row][0] + " "); // 0열 출력
            System.out.print(arr[row][1] + " "); // 1열 출력
            System.out.print(arr[row][2] + " "); // 2열 출력
            System.out.println();                // 한 행이 끝나면 라인 변경
        }
    }
}
```

* `row = 0`일 때 `arr[0][0], arr[0][1], arr[0][2]` → **1 2 3**
* `row = 1`일 때 `arr[1][0], arr[1][1], arr[1][2]` → **4 5 6**

#### 실행 결과

```
1 2 3 
4 5 6 
```

# 구조 개선(계속) - 열 출력 반복

다음 부분도 **열 인덱스(0, 1, 2)** 만 달라지며 같은 코드가 반복됩니다.

```java
System.out.print(arr[row][0] + " "); // 0열 출력
System.out.print(arr[row][1] + " "); // 1열 출력
System.out.print(arr[row][2] + " "); // 2열 출력
```

따라서 **열(column)** 역시 변수로 두고 **중첩 반복**으로 바꿀 수 있습니다.

```java
// column을 0, 1, 2로 변경하면서 다음 코드를 반복
System.out.print(arr[row][column] + " "); // column열 출력
```

### ArrayDi2

```java
package array;

public class ArrayDi2 {
    public static void main(String[] args) {
        // 2x3 2차원 배열을 만듭니다.
        int[][] arr = new int[2][3]; // 행(row), 열(column)

        arr[0][0] = 1; // 0행, 0열
        arr[0][1] = 2; // 0행, 1열
        arr[0][2] = 3; // 0행, 2열
        arr[1][0] = 4; // 1행, 0열
        arr[1][1] = 5; // 1행, 1열
        arr[1][2] = 6; // 1행, 2열

        for (int row = 0; row < 2; row++) {            // 바깥 반복: 행
            for (int column = 0; column < 3; column++) { // 안쪽 반복: 열
                System.out.print(arr[row][column] + " ");
            }
            System.out.println(); // 한 행이 끝나면 라인 변경
        }
    }
}
```

> 위 내부 `for`는, 이전의 아래 3줄을 **한 덩어리**로 대체합니다.

```java
System.out.print(arr[row][0] + " "); // 0열 출력 
System.out.print(arr[row][1] + " "); // 1열 출력 
System.out.print(arr[row][2] + " "); // 2열 출력
```
