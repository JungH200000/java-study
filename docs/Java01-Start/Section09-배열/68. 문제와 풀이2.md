## 문제 - 가장 작은 수, 큰 수 찾기 (`array.ex.ArrayEx6`)

- 사용자로부터 n개의 정수를 입력받아 배열에 저장한 후, 배열 내에서 가장 작은 수와 가장 큰 수를 찾아 출력하는 프로그램을 작성하자. 실행 결과 예시를 참고하자.

### 실행 결과 예시:

```terminaloutput
입력받을 숫자의 개수를 입력하세요:3 
3개의 정수를 입력하세요:
1
2
5
가장 작은 정수: 1 
가장 큰 정수: 5
```

## 문제 - 2차원 배열1 (`array.ex.ArrayEx7`)

사용자로부터 4명 학생의 국어, 수학, 영어 점수를 입력받아 각 학생의 총점과 평균을 계산하는 프로그램을 작성하자. 2차원 배열을 사용하고, 실행 결과 예시를 참고하자.

### 실행 결과 예시:

```terminaloutput
1번 학생의 성적을 입력하세요:
국어 점수:100
영어 점수:80
수학 점수:70
2번 학생의 성적을 입력하세요:
국어 점수:30
영어 점수:40
수학 점수:50
3번 학생의 성적을 입력하세요:
국어 점수:60
영어 점수:70
수학 점수:50
4번 학생의 성적을 입력하세요:
국어 점수:90
영어 점수:100
수학 점수:80
1번 학생의 총점: 250, 평균: 83.33333333333333
2번 학생의 총점: 120, 평균: 40.0
3번 학생의 총점: 180, 평균: 60.0
4번 학생의 총점: 270, 평균: 90.0
```
