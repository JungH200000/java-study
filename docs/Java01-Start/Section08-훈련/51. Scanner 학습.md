# Scanner 학습

## 1) 훈련 시작

지금까지 학습한 변수, 연산자, 조건문, 반복문은 프로그래밍의 **가장 기본**이 되는 기능이다. 대부분의 프로그램 언어는 이 기능을 필수로 가진다. 그리고 프로그래머가 하는 일의 대부분은 지금까지 설명한 변수, 연산자, 조건문, 반복문을 다 루는 일이다. 그래서 이 기능을 잘 다루는 것이 무엇보다 중요하다.

이번 시간에는 **지금까지 배운 내용들을 훈련하는 시간**이다. 여러분이 다음으로 나아가기 전에 최소한의 기본기를 훈련 하는 시간으로 생각하자

지금까지 학습할 때 한가지 아쉬움이 있었는데, 바로 사용자의 입력이 없었다는 점이다.
이번 시간에는 사용자의 입력을 받는 방법을 배워서, **좀 더 그럴듯한 프로그램**을 만들어보자.

---

## 2) Scanner 소개 (System.in 입력을 쉽게)

`System.out`이 출력이라면, 입력은 `System.in`이다. 다만 `System.in`을 직접 다루면 **복잡**하므로, 자바는 이를 쉽게 해 주는 **`Scanner` 클래스**를 제공한다.
아래 예제처럼 `new`로 `Scanner`를 만들어 사용한다. (객체/클래스는 뒤에서 자세히 학습)

---

## 3) 예제 1: `Scanner1`

```java
package scanner;

import java.util.Scanner;

public class Scanner1 {
    public static void main(String[] args) {
        // System.in(표준 입력)을 읽는 Scanner 생성
        Scanner scanner = new Scanner(System.in);

        System.out.print("문자열을 입력하세요:");
        String str = scanner.nextLine(); // 엔터(\n) 전까지 한 줄 전체를 문자열로 읽음
        System.out.println("입력한 문자열: " + str);

        System.out.print("정수를 입력하세요:");
        int intValue = scanner.nextInt(); // 정수를 int로 읽음
        System.out.println("입력한 정수: " + intValue);

        System.out.print("실수를 입력하세요:");
        double doubleValue = scanner.nextDouble(); // 실수를 double로 읽음
        System.out.println("입력한 실수: " + doubleValue);
    }
}
```

### 라인별 핵심 포인트

* `new Scanner(System.in)`: 표준 입력을 읽는 **스캐너 객체** 생성.
* `nextLine()`: **엔터(\n)** 를 만날 때까지 한 줄을 **문자열**로 읽음.
* `nextInt()`: **정수**를 `int`로 읽음.
* `nextDouble()`: **실수**를 `double`로 읽음.
* `print()`로 프롬프트를 띄우고, 입력을 받은 뒤 `println()`으로 결과를 한 줄씩 출력.

타입이 다르면 **오류**가 발생한다. 예를 들어 정수 자리(숫자)에 문자를 넣으면 다음과 같이 예외가 난다.

## 6) `print()` vs `println()`

* **`print()`**: 출력 **후 줄바꿈 없음** → 다음 출력이 **같은 줄**에 이어짐.

  ```java
  System.out.print("hello");
  System.out.print("world");
  // 결과: helloworld
  ```
* **`println()`**: 출력 **후 줄바꿈(\n)** → 다음 출력이 **다음 줄**로 내려감.

  ```java
  System.out.println("hello");
  System.out.println("world");
  // 결과:
  // hello
  // world
  ```
* 우리가 엔터 키를 치면 `\n`(newline, 개행 문자)이 남고, 콘솔은 이를 보고 **다음 줄로 이동**한다.
  아래 코드는 `println()`과 동일한 효과다.

  ```java
  System.out.print("hello\n");
  System.out.print("world\n");
  // 결과:
  // hello
  // world
  ```
