# Scanner - 기본 예제

**목표:** `Scanner`를 사용해 **두 정수의 합을 출력**하고, **두 정수 중 큰 값을 판별**하는 흐름을 연습한다.

---

## 예제 2: 두 수의 합 구하기 (`Scanner2`)

```java
package scanner;

import java.util.Scanner;

public class Scanner2 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in); // 표준 입력을 읽는 Scanner 생성

        System.out.print("첫 번째 숫자를 입력하세요:"); // 줄바꿈 없이 프롬프트 출력
        int num1 = scanner.nextInt();              // 첫 번째 정수 입력

        System.out.print("두 번째 숫자를 입력하세요:"); // 다음 정수 입력 안내
        int num2 = scanner.nextInt();               // 두 번째 정수 입력

        int sum = num1 + num2;                      // 합 계산
        System.out.println("두 숫자의 합: " + sum); // 결과 출력
    }
}
```

### 실행 결과

```
첫 번째 숫자를 입력하세요:10
두 번째 숫자를 입력하세요:20
두 숫자의 합: 30
```

## 예제 3: 더 큰 수 출력하기 (`Scanner3`)

```java
package scanner;

import java.util.Scanner;

public class Scanner3 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);   // Scanner 준비

        System.out.print("첫 번째 숫자를 입력하세요:"); 
        int num1 = scanner.nextInt();               // 첫 번째 정수

        System.out.print("두 번째 숫자를 입력하세요:"); 
        int num2 = scanner.nextInt();               // 두 번째 정수

        // 조건문으로 크기 비교: >, <, == 중 하나의 분기만 실행
        if (num1 > num2) {
            System.out.println("더 큰 숫자: " + num1);
        } else if (num2 > num1) {
            System.out.println("더 큰 숫자: " + num2);
        } else {
            System.out.println("두 숫자는 같습니다.");
        }
    }
}
```

### 실행 결과

```
첫 번째 숫자를 입력하세요:10
두 번째 숫자를 입력하세요:20
더 큰 숫자: 20
첫 번째 숫자를 입력하세요:5
두 번째 숫자를 입력하세요:5
두 숫자는 같습니다.
```

---

## 용어 한 줄 정리

* **Scanner**: 표준 입력(`System.in`)에서 값을 읽는 도구.
* **`nextInt()`**: 입력을 **정수(int)** 로 읽는다.
* **조건문(`if-else if-else`)**: 주어진 **조건에 따라 실행 경로**를 선택한다.
