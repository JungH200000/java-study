# 변수와 초기화

## 1. 변수의 종류

자바에서 변수는 크게 **선언 위치**에 따라 다음 두 가지로 나눌 수 있다.

* **멤버 변수(필드)**: 클래스에 선언되는 변수
* **지역 변수**: 메서드 내부에 선언되는 변수

    * **매개변수(파라미터)** 도 지역 변수의 한 종류이다.

---

### 1.1 멤버 변수(필드) 예시

```java
public class Student {
    String name;
    int age;
    int grade;
}
```

* `name`, `age`, `grade` 는 **멤버 변수(필드)** 이다.
* 클래스 `Student` 안에 선언되어 있으므로, 각 `Student` 인스턴스가 이 필드들을 가진다.

---

### 1.2 지역 변수 예시

```java
public class ClassStart3 {
    public static void main(String[] args) {
        Student student1;
        student1 = new Student();

        Student student2 = new Student();
    }
}
```

* `student1`, `student2` 는 **지역 변수**이다.
* 둘 다 `main()` 메서드 블록 안에서 선언되었기 때문이다.

---

### 1.3 지역 변수와 매개변수 예시

```java
public class MethodChange1 {
    public static void main(String[] args) {
        int a = 10;
        System.out.println("메서드 호출 전: a = " + a);
        changePrimitive(a);
        System.out.println("메서드 호출 후: a = " + a);
    }

    public static void changePrimitive(int x) {
        x = 20;
    }
}
```

* `a`, `x(매개변수)` 는 모두 **지역 변수**이다.

지역 변수는 이름 그대로 **특정 지역(블록)** 에서만 사용되는 변수이다.

* 변수 `x` 는 `changePrimitive()` 메서드의 블록 안에서만 사용할 수 있다.
  `changePrimitive()` 메서드가 끝나면 `x` 는 제거된다.
* 변수 `a` 도 마찬가지로 `main()` 메서드가 끝나면 제거된다.

---

## 2. 변수의 값 초기화

### 2.1 멤버 변수: 자동 초기화

**인스턴스의 멤버 변수는 인스턴스를 생성할 때 자동으로 초기화**된다.

* 숫자(`int`)   → `0`
* `boolean`     → `false`
* 참조형        → `null`

    * `null` 값은 **참조할 대상이 없다는 뜻**으로 사용된다.

또한, **개발자가 초기값을 직접 지정할 수도 있다.**

---

### 2.2 지역 변수: 수동 초기화

* **지역 변수는 항상 직접 초기화해야 한다.**
* 자동으로 기본값이 들어가지 않으므로, 사용 전에 값을 꼭 넣어야 한다.

---

## 3. 멤버 변수 초기화 예제

### 3.1 `InitData` 클래스

```java
package ref;

public class InitData {
    int value1;        // 초기화 하지 않음
    int value2 = 10;   // 10으로 초기화
}
```

* `value1` 은 **초기값을 지정하지 않은 멤버 변수**이고,
* `value2` 는 **초기값을 10으로 지정한 멤버 변수**이다.

---

### 3.2 `InitMain` 클래스

```java
package ref;

public class InitMain {
    public static void main(String[] args) {
        InitData data = new InitData();
        System.out.println("value1 = " + data.value1);
        System.out.println("value2 = " + data.value2);
    }
}
```

#### 실행 결과

```text
value1 = 0
value2 = 10
```

설명:

* `value1` 은 초기값을 지정하지 않았지만, **멤버 변수는 자동으로 초기화**되므로
  숫자인 `value1` 은 `0` 으로 초기화된다.
* `value2` 는 코드에서 **10으로 초기값을 지정**해 두었기 때문에,
  객체를 생성할 때 `10` 으로 초기화된다.
